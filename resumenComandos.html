Inicializar Git

Inicializa un nuevo repositorio Git en tu proyecto:

git init

Trabajar con Ramas

Crear una nueva rama y cambiar a ella:

git branch nombre-rama
git checkout nombre-rama

Cambiar a una rama existente:

git checkout nombre-rama

Unir una rama al main:

git checkout main
git merge nombre-rama

Clonar Repositorio

Clonar un repositorio existente desde GitHub:

git clone URL-del-repositorio

Subir Cambios

Subir los cambios al repositorio remoto:

git add .
git commit -m "Descripción del cambio"
git push origin nombre-rama

Instalación de npm

Instalar las dependencias del proyecto:

npm install

Iniciar Proyecto

Ejecutar el servidor de desarrollo de React:

npm run dev

Construir Proyecto

Crear una versión optimizada para producción:

npm run build
